# Editor

This section is a reference manual for the Godot editor itself — not game design, not programming concepts, just "where is this button and what does it do." If you've never opened Godot before, this is where to start.

![The Scene dock](../assets/images/scene_dock.png)

## If you're just getting started

Open Godot, create a new scene ([Scenes & Nodes](scenes-and-nodes/new-scene.md)), and attach your first script ([Scripting Basics](scripting-basics/create-script.md)). Everything in [Building Games](../building-games/scene-composition/ground.md) and [Techniques](../techniques/index.md) assumes you can already do both of those things.
